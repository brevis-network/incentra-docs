# Get Campaigns

Incentra provides restful APIs to access the campaign data.

### Core Functionality

The SDK provides details about different types of incentive campaigns launched on Incentra:

1. [**Concentrated Liquidity Campaigns**](concentrated-liquidity-campaign.md) - Incentivize providing concentrated liquidity in DEXes.
2. [**Token Holding Campaigns**](token-holding-campaign.md) - Incentivize holding specific tokens.
3. [**Lend and Borrow Campaigns (Euler)** ](lend-and-borrow-campaign-euler.md)- Incentivize lending or borrowing from Euler finance.
